Welcome to the Transformers API

Your place for all your transformers data needs

Autobot Routes

  • GET /api/autobots - Retrieve all Autobots
  • GET /user - Retireve a sing user and their associated autobots Requires Authentication
  • POST /api/autobots - Add an Autobot to the database JSON {name: STRING, color: STRING}
  • PUT /api/autobot - Update an Autobot in the database JSON {autobot_id: ID name: STRING, color: STRING} | Requires authentication
  • DELETE /api/autobot - Delete an Autobot from the database JSON {autobot_id: ID} | Requires authentication

User Routes

  • POST /api/auth/register - Register a user account JSON {email: STRING, password: STRING (minimum 6 charaters)}
  • POST /api/auth/login - Log In to your account JSON {email: STRING, password: STRING}
